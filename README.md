# Villa Butterfly Marrakech Website

Professional static website for Villa Butterfly luxury rental in Amelkis Golf Resort, Marrakech.

## Key Features
- **Multilingual**: English primary, with French and Spanish options (basic switcher implemented).
- **SEO Optimized**: Meta tags, schema markup for LodgingBusiness, canonical URL.
- **Trust Building**: Real guest reviews, direct owner contact, VVVIP focus.
- **Responsive Design**: Modern luxury style combining Marrakech classic elements.
- **Contact Form**: Simple form linking to Google Sheets (replace YOUR_FORM_ID with actual Google Form ID).
- **WhatsApp Direct**: Link to owner's WhatsApp for immediate contact.
- **Real Media**: Gallery with actual villa photos and videos provided.
- **Logo**: Placeholder for butterfly logo (create butterfly-logo.png with villa's design).
